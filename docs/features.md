# Features

Packistry is a self-hosted Composer repository for PHP package distribution.

## Private Repository Support

Host proprietary packages in private repositories with controlled access.

## Token-Based Authentication

Use token-based access for humans and machines with granular permissions.

## Package Source Integration

Import and sync packages from:

- GitHub
- GitLab
- Gitea
- Bitbucket

Packistry keeps imported sources up to date using webhooks.

## Manual ZIP Uploads

Upload package ZIP archives directly without configuring a source provider.

## Repository Management

- Public and private repository modes
- User-based access control for people
- Deploy token support for automation and CI/CD

## Authentication Sources

Packistry supports SSO via OAuth 2.0 and OpenID Connect.

- OpenID Connect (for providers such as Okta or Microsoft Entra ID)
- GitHub
- GitLab
- Bitbucket
- Google
